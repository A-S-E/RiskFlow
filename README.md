# RiskFlow Pro: Quantitative Portfolio Analytics

RiskFlow Pro is a full-stack financial analytics engine designed to help investors understand portfolio risk through advanced quantitative metrics. The application fetches real-time market data to calculate volatility, market sensitivity (Beta), and diversification potential.

## 🚀 Key Features
- **Portfolio Persistence:** Save and manage multiple portfolios using a custom SQLite database.
- **Risk Metrics:** Real-time calculation of Annualized Volatility, Portfolio Beta, and Sharpe Ratio.
- **Predictive Analytics:** Daily Value at Risk (VaR) at a 95% confidence interval.
- **Diversification Analysis:** Dynamic Correlation Matrix with a heatmap visualization to identify asset relationships.
- **Interactive Visuals:** Comparative growth charts against the S&P 500 benchmark.

## 🛠️ Tech Stack
- **Frontend:** React, Recharts, Lucide Icons, Axios
- **Backend:** Python, FastAPI, Uvicorn
- **Data Science:** Pandas, NumPy, yfinance
- **Database:** SQLAlchemy, SQLite

## 📊 Analytics Deep Dive
- **Sharpe Ratio:** Measures risk-adjusted return, helping users determine if their gains are worth the price of volatility.
- **Beta:** Quantifies the portfolio's systemic risk relative to the broader market.
- **Correlation Matrix:** Uses a color-coded heatmap to visualize how assets move in relation to one another.
